# E-Commerce-App

__Description:__ Provides e-commerce services. Admin can maintian products, Customers will be able to buy products from home.

*There are 3 type of users - Customers & Admin & Sellers*

### Customer:

* Register & Login
* Reset password answering security questions 
* Search products 
* Add to cart and place order
* Change profile

### Admin: (Belongs to the app organization)

* Register & Login
* Add new products in various categories
* Approve products uploaded by seller
* Maintain products such as edit, delete
* Check new orders and confirm orders

### Seller: (General person who wants to sell his products)

* Register & Login
* Add new products
* Check new orders

### Database Structure:

* Admins
* Products
* Sellers
* Users
* Profile Images
* Products Images
* Cart Lists
